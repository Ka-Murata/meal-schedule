鶏照り焼きと彩り野菜の丼1人分

材料

  • ご飯180g
  • 鶏もも肉120g
  • 玉ねぎ50g
  • ピーマン40g
  • にんじん30g
  • しょうゆ15ml
  • みりん15ml
  • 砂糖5g
  • サラダ油5ml

手順

  1. 翌日分と合わせて材料を準備し、鶏肉は一口大、玉ねぎは薄切り、ピーマンとにんじんは細切りにする。
  2. 油を熱したフライパンで鶏肉を皮目から4分、返して2分焼き、野菜を加えて3分炒める。
  3. しょうゆ、みりん、砂糖を加え、汁気が少なくなるまで2〜3分絡める。半量を食べ、半量は粗熱を取ってすぐ冷蔵する。

ツナと小松菜の塩昆布混ぜご飯1人分

材料

  • ご飯180g
  • ツナ水煮缶1缶
  • 小松菜80g
  • 塩昆布8g
  • 白いりごま3g
  • ごま油5ml

手順

  1. 小松菜を2cm幅に切り、耐熱容器に入れて600Wで1分30秒加熱する。
  2. 水気を絞り、汁気を切ったツナ、塩昆布、白ごま、ごま油と和える。
  3. 温かいご飯に具を混ぜて盛る。

鮭ときのこの味噌バター焼き1人分

材料

  • ご飯180g
  • 生鮭1切れ
  • しめじ70g
  • キャベツ100g
  • 味噌15g
  • みりん10ml
  • バター5g
  • 黒こしょう少々

手順

  1. 翌日分と合わせ、しめじをほぐし、キャベツを3cm角に切る。味噌とみりんを混ぜる。
  2. フライパンにキャベツ、しめじ、鮭を置き、味噌だれとバターをのせる。水大さじ2を加えて蓋をする。
  3. 中火で2分、弱火で8〜10分蒸し、鮭の中心まで火を通す。弁当分は骨を除き、粗熱を取ってすぐ冷蔵する。

鶏肉と夏野菜のトマトカレー1人分

材料

  • ご飯180g
  • 鶏むね肉120g
  • なす80g
  • ズッキーニ70g
  • 玉ねぎ50g
  • カットトマト缶100g
  • カレー粉5g
  • 顆粒コンソメ3g
  • サラダ油5ml
  • 水60ml

手順

  1. 翌日分と合わせ、鶏肉、なす、ズッキーニを2cm角、玉ねぎを粗みじんに切る。
  2. 油で鶏肉を3分炒め、野菜を加えて4分炒める。カレー粉を加え30秒炒める。
  3. トマト、コンソメ、水を加え、蓋をずらして12分煮る。弁当分は浅い容器で素早く冷まし、冷蔵する。

さばと大根の味噌煮1人分

材料

  • ご飯180g
  • さば切り身1切れ
  • 大根150g
  • しょうが8g
  • 味噌18g
  • 砂糖6g
  • 酒30ml
  • みりん15ml
  • 水120ml

手順

  1. 大根を1cm厚さのいちょう切り、しょうがを薄切りにし、さばへ熱湯を回しかける。
  2. 鍋に水、酒、みりん、砂糖、大根、しょうがを入れて沸かし、さばを皮目を上にして加える。
  3. 落とし蓋をして12分煮る。煮汁で味噌を溶いて加え、さらに5分煮て中心まで火を通す。
